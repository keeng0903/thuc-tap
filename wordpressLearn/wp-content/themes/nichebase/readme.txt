== Description ==
NicheBase is a minimal and fully functional WordPress theme developed with the intention of combining all of your business solutions in a single theme. It's highly customizable options with a clean and elegant design is perfect to create all kind of websites. Flexible framework combined with the Elementor page builder plugin optimizes your business growth by more productivity with less work.

Main features include

* Customizable colors (header, content, footer)
* Fully Responsive
* 4 Footer Columns
* Blog Sidebar Position Change
* And More...

== Installation ==

1. In your WordPress Dashboard, go to Appearance > Themes and click the "Add New" button.
2. Type "NicheBase" in the search form and search it.
3. Click the "Activate" button to use it.
4. Navigate to Appearance > Customize in your WordPress Dashboard and Customize it.

== Frequently Asked Questions ==

= This theme runs on the latest version of WordPress? =
Yes, Nichebase themes run on the newest version of WordPress. You can confirm it from "WordPress Version" information from this page.

= How do I install the theme? =
To install any theme on your site is, you must first have a working WordPress installation. Once WordPress has been properly set up, Goto Appearance > Themes > Add New and Search for "NicheBase" there you can find and click install and activate it. That's All!
